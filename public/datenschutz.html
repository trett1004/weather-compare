Diese Website speichert die von Ihnen gespeicherten Orte ausschließlich lokal im localStorage Ihres Browsers. Diese Daten verlassen Ihr Gerät nicht und werden nicht an unsere Server übertragen. Es handelt sich um technisch notwendige Funktionalität, die keiner Einwilligung bedarf.

Sie können diese Daten jederzeit löschen, indem Sie den Browser-Cache leeren (Entwicklerwerkzeuge → Application → Local Storage).
